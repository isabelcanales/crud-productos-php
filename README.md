# CRUD de Productos en PHP

Aplicación web desarrollada en **PHP utilizando PDO y MySQL** que permite gestionar productos mediante operaciones CRUD.

## Funcionalidades

- Crear nuevos productos
- Visualizar listado de productos
- Actualizar información de productos
- Eliminar productos existentes

## Tecnologías utilizadas

- PHP
- MySQL
- PDO (PHP Data Objects)
- HTML
- CSS

## Estructura del proyecto

- `index.php` → Página principal de navegación
- `crear.php` → Inserción de nuevos productos
- `leer.php` → Listado de productos
- `actualizar.php` → Modificación de productos
- `borrar.php` → Eliminación de productos
- `db.php` → Conexión a la base de datos
- `estilos.css` → Estilos de la interfaz

## Descripción

Este proyecto fue desarrollado como práctica del ciclo formativo **Desarrollo de Aplicaciones Multiplataforma (DAM)** y tiene como objetivo aplicar conceptos de desarrollo web con PHP y bases de datos.
